== Description ==

A plugin that creates a shortcode that show a random Bible verse from the King James Version (KJV)

Use the shortcode [mcm-random-bible-verse-wp] 

Note: all Bible verses are stored directly in the database, so no external service is used to retrieve them.

Thanks to apostolic-churches.net for formatting the KJV Bible in a format we could parse into the database used for this plugin.


== Installation ==

1. Unzip the plugin zip file
2. Upload the entire folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. The shortcode is [mcm-random-bible-verse-wp]
